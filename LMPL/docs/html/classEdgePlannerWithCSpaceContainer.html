Detailed Description

Convenience class for edge planner that holds a smart pointer to a CSpace. Typically used for single-obstacle edge checkers as follows:

SingleObstacleCSpace* ospace = new SingleObstacleCSpace(this,obstacle) return new EdgePlannerWithCSpaceContainer(ospace,new XEdgePlanner(ospace,a,b))
