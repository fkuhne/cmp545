Detailed Description

A grid-based subdivision to be used for SBL.

The grid operates on certain dimensions of the configuration. Specifically, it picks mappedDims dimensions at random from the full configuration space, and divides the space in those dimensions into cells of width h. If the mapped dimension is that of index k.

NOTE: h must be initialized to the # of dims in the configuration space, and containing some cell width value, say 0.1.
