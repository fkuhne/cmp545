Detailed Description

The same as StraightLineEpsilonPlanner, but keeps the bisected configs, and recalculates distances for every subdivision.

Used in constrained configuration spaces where CSpace.Midpoint() doesn't necessarily return a configuration whose distance is half of the endpoints.
